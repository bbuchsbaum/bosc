# Non-parametric p-values vs reference

Non-parametric p-values vs reference

## Usage

``` r
nonparam_pval(data, ref, alpha = 0.05)
```

## Arguments

- data:

  numeric vector.

- ref:

  numeric vector of reference samples.

- alpha:

  significance level.

## Value

list with `h` significance (0/1) and `p` p-values.
