# Pairwise Phase Consistency (1D)

Bias-free PPC for angle data along a chosen dimension.

## Usage

``` r
pairwise_phase_consistency(data, dim = 1)
```

## Arguments

- data:

  array of angles (radians).

- dim:

  integer; dimension along which to compute PPC.

## Value

array of PPC values with the chosen dimension collapsed.
