# bosc: Behavioral Oscillation Scores and Phase Analysis

R port of the behavioral-oscillations MATLAB toolbox, providing
functions for oscillation scoring, surrogate testing, phase analyses,
PPC metrics, clustering, and visualization for behavioral and neural
time series.
